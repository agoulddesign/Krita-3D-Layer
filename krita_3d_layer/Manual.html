Krita 3D Layer Plugin

Welcome to the Krita 3D Layer Plugin! This tool allows you to seamlessly integrate 3D models into your 2D painting workflow by managing them as synchronized layers within Krita.

✨ Key Features

Layer-Linked 3D State

Import and position your 3D model and click "Apply Layer". The plugin bakes the 3D state (model path, camera angles, rotations) into the layer registry, allowing you to return to any previous 3D setup instantly.

Orbit & Walk Camera Modes

Freely switch between standard Orbit navigation and immersive first-person Walk controls (mouse to look, scroll to move).

Generative 3D (ComfyUI)

Direct integration with ComfyUI allows you to generate 3D models directly from your Krita canvas drawings using Image-to-3D workflows.

🚀 Standard Installation

  1. Download the latest release .zip file.
  2. In Krita, go to Tools -> Scripts -> Import Python Plugin from File...
  3. Select the downloaded .zip file and click OK.
  4. Restart Krita.
  5. Enable the plugin in Settings -> Configure Krita... -> Python Plugin Manager.
  6. Restart Krita once more. The Docker is available under Settings -> Dockers -> 3D Layer.

Note: If installing via Git clone, you must manually ensure PyOpenGL is available in your environment.

🎮 Basic Usage

Local 3D Models

  • LMB: Rotate (Orbit) / Look (Walk)
  • MMB: Pan (Orbit) / Strafe (Walk)
  • Scroll: Zoom (Orbit) / Move Forward/Back (Walk)
  • Shift: Precision mode for all movements.

Generative Workflow

  1. Draw your subject on a layer with a transparent background.
  2. Set your ComfyUI server address in the Generate tab.
  3. Load your API-formatted workflow JSON.
  4. Click Generate 3D Model.

📦 Technical details

This plugin uses the Krita Python API and PyQt5 for the interface. 3D rendering is handled by a custom hardware-accelerated OpenGL viewport. It is designed to be lightweight and portable.
